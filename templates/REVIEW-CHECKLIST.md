# Review Checklist

Run these reviews with Claude. Open this file in a Claude session, say "let's do the weekly review" (or monthly / quarterly), and walk through it together.

## Weekly Review (~20 min)

### 1. Active Projects Check
- [ ] Read PROJECT-INDEX.md — are the active projects still the right ones?
- [ ] For each active project: What did I accomplish this week? What's blocking me?
  - [ ] **[Project 1]:**
  - [ ] **[Project 2]:**
  - [ ] **[Project 3]:**
- [ ] Am I over the 5-active-project limit? If yes, what gets paused?

### 2. Focus Check
- [ ] Did I stick to one project per session this week?
- [ ] How many context switches happened? Was it costly?
- [ ] Did I fall into any rabbit holes?

### 3. Pipeline Check
- [ ] Any new ideas added to IDEAS.md this week?
- [ ] Any ideas past their 7-day cooling period ready to score?
- [ ] Anything in PIPELINE.md ready to promote or kill?

### 4. Quick Wins
- [ ] Any small tasks avoided that would take < 30 minutes?
- [ ] Any project that needs a status change in PROJECT-INDEX.md?

### 5. Next Week
- [ ] What's the #1 thing to accomplish next week?
- [ ] Any deadlines or time-sensitive items?

---

## Monthly Review (~45 min)

*Do the weekly review first.*

### 6. Project Health
- [ ] For each active project: Is it making real progress toward a milestone, or churning?
- [ ] For each paused project: Has its resume trigger been met?
- [ ] Any low-priority projects to promote or kill?

### 7. Revenue and Business
- [ ] Any revenue this month? From what?
- [ ] Are the current active projects the best path to revenue?

### 8. Strategy Check
- [ ] Re-read the strategic plan — still aligned?
- [ ] Has the market changed for any active projects?
- [ ] Any new opportunities worth adding to IDEAS.md?

### 9. Pipeline Review
- [ ] Walk through all IDEAS.md entries. Any stale? Any newly compelling?
- [ ] Walk through PIPELINE.md. Score anything unscored.
- [ ] Is anything in the pipeline stronger than an active project?

### 10. Update Documents
- [ ] Update PROJECT-INDEX.md with status changes
- [ ] Update PRIORITIES.md
- [ ] Update "Last updated" dates

---

## Quarterly Review (~1 hr)

*Do the monthly review first.*

### 11. Big Picture
- [ ] Where am I relative to the annual strategic plan?
- [ ] What worked well this quarter? What didn't?
- [ ] Am I still excited about the direction?

### 12. Capacity and Sustainability
- [ ] Is the pace sustainable?
- [ ] Should the max active projects limit change?
- [ ] Review research/ docs — any research to refresh?
